# memory

> 我们可以共享所有记忆的最佳存储空间

## Build Setup

``` bash
# install dependencies
$ npm run install

# rename .env-sample to .env
mv .env-sample .env

# set environment variables
vim .env

# initialize the database
npm run init-db

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
